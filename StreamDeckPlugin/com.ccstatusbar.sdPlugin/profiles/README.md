# Stream Deck Profiles

This directory contains pre-configured profiles for different Stream Deck devices.

## Creating Profiles

1. Open Stream Deck app
2. Configure your layout with CC Status Bar actions
3. Right-click profile name â†’ Export Profile
4. Save to this directory with names matching manifest.json:
   - `CCStatusBar Standard.streamDeckProfile`
   - `CCStatusBar Mini.streamDeckProfile`
   - `CCStatusBar XL.streamDeckProfile`

## Recommended Layouts

### Mini (6 buttons, 2x3)
```
[S1] [S2] [S3]
[Up] [Dn] [Enter]
```

### Standard (15 buttons, 3x5)
```
[S1] [S2] [S3] [S4] [Up]
[S5] [S6] [S7] [S8] [Dn]
[S9] [S10][Esc][Mic][Enter]
```

### XL (32 buttons, 4x8)
```
[S1] [S2] [S3] [S4] [S5] [S6] [S7] [S8]
[S9] [S10][S11][S12][S13][S14][S15][S16]
[   ][   ][   ][   ][   ][   ][   ][   ]
[Up] [Dn] [Esc][Mic][Enter][  ][  ][  ]
```
